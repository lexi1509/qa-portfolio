## TC-002: Блокировка личного кабинета при пятикратном неправильном вводе пароля

| Поле | Значение |
|------|----------|
| **ID** | TC-002 |
| **Заголовок** | Блокировка личного кабинета при пятикратном неправильном вводе пароля |
| **Предусловия** | Пользователь зарегистрирован и не заблокирован |
| **Шаги** | 1. Ввести логин: `multicustomer`<br>2. Ввести неверный пароль (повторить 5 раз с интервалом 3–5 сек)<br>3. Нажимать «Войти» после каждой попытки |
| **Тестовые данные** | Пароли: `wb6u4Ao`, `PpFN1ZH`, `dulHpYD`, `8YluHQ2`, `nOyCtCR` |
| **Ожидаемый результат** | После 5-й попытки система блокирует учётную запись и сообщает период блокировки (ММ:СС) |
| **Фактический результат** | Блокировка не происходит, вход остаётся доступен |
| **Статус** | ❌ FAIL |
| **Связанный баг** | [BR-001](ссылка на баг-репорт) |
| **Окружение** | Яндекс.Браузер 25.8.5.983 (64-bit), Windows 10 22H2 |
